Random Number Guessing

This project is a console-based Number Guessing Game implemented in C#. The game generates a random number between 1 and 100 using the `Random` class, and the player must guess the number within this range. The purpose of the game is to provide an interactive and engaging way to practice programming concepts such as loops, conditionals, and user input handling in C#. Key features include a welcome message, input validation to ensure guesses are within the valid range, and feedback on whether the guesses are too high or too low. The game keeps track of the number of attempts and congratulates the player when they successfully guess the number. Technologies used in this project are the C# programming language and the .NET framework for creating the console application. This project serves as an educational tool for beginners to learn and practice basic programming skills.
